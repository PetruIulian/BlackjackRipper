The following classes exist within the project:
* **sX**, **kX**, **hX**, **rX** where **s**, **k**, **h**, and **r** stand for schoppen, klaver, harten, ruiten respectively. The **X** can be the numbers **2-9**, **a** for an ace, or **b**, **v** or **h** for boer, vrouw and heer respectively.
* The **j** class stands for joker.
* The **pile-facing-up** and **pile-facing-down** classes should be used to label piles
